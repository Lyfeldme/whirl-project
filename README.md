A modern landing page for **Whirl** — a service that helps you automate your everyday tasks and workflows. This project focuses on clean design, responsiveness, and interactive user experience.

## ✨ Features

- **📱 Fully Responsive Design** — perfect display on all devices
- **🌗 Theme Switcher** — toggle between light and dark mode with one click
- **📦 Modal Windows** — sign in, sign up, password recovery, demo booking
- **🎨 SVG Sprites** — optimized graphics and icons
- **⚡ Interactive Elements** — custom accordion, checkboxes, and forms

## 🚀 Functionality

### Modal Windows
- 🔐 **Sign In** — user authentication
- 📝 **Sign Up** — new account registration
- 🔑 **Forgot Password** — password recovery flow
- 📅 **Book a Demo** — schedule a product presentation
- ℹ️ **Info Modals** — for "Learn more" and "See All Posts" buttons

### Interactivity
- 🎯 Accordion-style dropdowns in the features section
- 💫 Smooth theme transition animations
- ✅ Form validation in all modals

## 🛠 Tech Stack

### Frontend
- **HTML5** — semantic markup
- **CSS3**
  - Modular stylesheet architecture
  - Flexbox & Grid layouts
  - Media queries for responsiveness
- **Vanilla JavaScript**
  - DOM manipulation
  - Event handling
  - Theme state management
